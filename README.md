# Backprop Canvas Landing Page

This is the fast static landing page for Backprop Canvas.

## Before publishing

Open `index.html` and replace every instance of:

```text
https://YOUR-HF-SPACE-URL.hf.space
```

with your actual Hugging Face Space direct URL.

Example:

```text
https://apoorvamehta-backprop-canvas.hf.space
```

## Deploy on GitHub Pages

1. Create a new GitHub repository, for example `backprop-canvas-site`.
2. Upload `index.html`.
3. Go to Settings → Pages.
4. Under Build and deployment, choose source: Deploy from a branch.
5. Choose branch: `main`, folder: `/root`.
6. Save.
